Шаг 1. Поскольку \(OM\perp AB\), точка \(M\) — середина хорды \(AB\), значит 
\[
AM=\frac{AB}{2}=\frac{6}{2}=3.
\]
Шаг 2. В прямоугольном треугольнике \(\triangle AOM\) по теореме Пифагора
\[
OA^2=OM^2+AM^2=4^2+3^2=25,
\]
откуда \(OA=OB=5\).
Шаг 3. Обозначим \(\alpha=\angle AOM\). Тогда в прямоугольном треугольнике
\[
\tan\alpha=\frac{AM}{OM}=\frac{3}{4}.
\]
Шаг 4. Угол \(AOB=2\alpha\). По формуле тангенса двойного угла
\[
\tan(2\alpha)=\frac{2\tan\alpha}{1-\tan^2\alpha}
=\frac{2\cdot\frac{3}{4}}{1-\bigl(\frac{3}{4}\bigr)^2}
=\frac{\frac32}{\frac7{16}}
=\frac{24}{7}\approx3{,}4286.
\]
Ответ: \(\tan\angle AOB\approx3{,}4286\).

Шаг 1. В окружности перпендикуляр к хорде всегда проходит через её середину. Это следует из свойства: «перпендикуляр, проведённый из центра окружности к хордe, делит хорду пополам». Поэтому если \(OM\) перпендикулярно \(AB\), то точка \(M\) является серединой отрезка \(AB\). Из условия \(AB=6\), значит половина хорды \(AM=\tfrac{6}{2}=3\).

Шаг 2. Рассмотрим треугольник \(AOM\). По построению он прямоугольный (угол при \(M\) равен \(90^\circ\)). Теорема Пифагора гласит: «в прямоугольном треугольнике квадрат гипотенузы равен сумме квадратов катетов». Здесь гипотенуза — \(OA\), а катеты — \(OM\) и \(AM\). Подставляем длины: \(OM=4\), \(AM=3\), получаем \(OA^2 = 4^2 + 3^2 = 16+9 = 25\). Значит \(OA = \sqrt{25} = 5\). Поскольку \(O\) — центр окружности, все радиусы равны, поэтому \(OB=5\) тоже.

Шаг 3. В прямоугольном треугольнике определение тангенса угла: «тангенс угла равен отношению противолежащего катета к прилежащему». Обозначим угол при вершине \(O\) на стороне \(OM\) как \(\alpha = \angle AOM\). Тогда \(\tan\alpha = AM/OM = 3/4\).

Шаг 4. Теперь угол между радиусами \(OA\) и \(OB\) есть угол \(AOB\), а он в два раза больше угла \(\alpha\): \(\angle AOB = 2\alpha\). Для работы с удвоенным углом используем формулу
\[
\tan(2\alpha) \;=\;\frac{2\tan\alpha}{1-\tan^2\alpha}.
\]
Подставляем \(\tan\alpha = 3/4\):
\[
\tan(2\alpha)
=\frac{2\cdot\frac{3}{4}}{1-\bigl(\frac{3}{4}\bigr)^2}
=\frac{\frac{3}{2}}{1-\frac{9}{16}}
=\frac{\frac{3}{2}}{\frac{7}{16}}
=\frac{3}{2}\times\frac{16}{7}
=\frac{24}{7}\approx3{,}4286.
\]
Это и есть \(\tan\angle AOB\). Окончательный ответ: приблизительно \(3{,}4286\).
