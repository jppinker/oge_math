Шаг 1. Найдём угол \(B\). По теореме о сумме внутренних углов треугольника
\[
\angle A + \angle B + \angle C = 180^\circ,
\]
следовательно
\[
\angle B = 180^\circ - 10^\circ - 64^\circ = 106^\circ.
\]

Шаг 2. В треугольнике \(ABH\) точка \(H\) лежит на стороне \(AC\), а \(BH\) — высота, значит \(BH\perp AC\). Перпендикулярные прямые образуют прямой угол:
\(\angle AHB = 90^\circ\). Поскольку \(H\) на \(AC\), отрезок \(AH\) лежит на той же прямой, что и \(AC\), то
\(\angle HAB = \angle CAB = 10^\circ\). Сумма углов в треугольнике \(ABH\) равна \(180^\circ\), откуда
\[
\angle ABH = 180^\circ - \angle HAB - \angle AHB = 180^\circ - 10^\circ - 90^\circ = 80^\circ.
\]

Шаг 3. \(BD\) — биссектриса угла \(B\), значит она делит угол \(\angle ABC\) пополам:
\[
\angle ABD = \frac{\angle B}{2} = \frac{106^\circ}{2} = 53^\circ.
\]

Шаг 4. Угол между прямыми \(BH\) и \(BD\) при вершине \(B\) равен разности углов, которые эти прямые образуют с лучом \(BA\). То есть
\[
\angle HBD = \bigl|\angle ABH - \angle ABD\bigr| = |80^\circ - 53^\circ| = 27^\circ.
\]

Ответ: \(27^\circ\).

solutiontextexpanded:
Шаг 1.  
В любом треугольнике три внутренних угла в сумме образуют полный развёрнутый угол, то есть \(180^\circ\). Это общеизвестная геометрическая теорема. В нашей задаче известны два угла: \(\angle A=10^\circ\) и \(\angle C=64^\circ\). Чтобы узнать третий угол, вычитаем их из \(180^\circ\):
\[
\angle B = 180^\circ - 10^\circ - 64^\circ = 106^\circ.
\]
Так мы нашли величину угла \(B\).

Шаг 2.  
Понятие «высота» в треугольнике означает отрезок, проведённый из вершины перпендикулярно противоположной стороне. Здесь \(BH\) — высота, она опущена на \(AC\), то есть \(BH\perp AC\). По определению перпендикулярных прямых угол между ними равен \(90^\circ\). Точка \(H\) лежит на отрезке \(AC\), значит отрезок \(AH\) полностью совпадает с частью прямой \(AC\). Угол при вершине \(A\) между \(AH\) и \(AB\) равен углу между \(AC\) и \(AB\), то есть \(\angle HAB = \angle CAB = 10^\circ\). В треугольнике \(ABH\) сумма углов равна \(180^\circ\), поэтому угол при вершине \(B\) этого треугольника вычисляется так:
\[
\angle ABH = 180^\circ - (\angle HAB + \angle AHB)
          = 180^\circ - (10^\circ + 90^\circ)
          = 80^\circ.
\]

Шаг 3.  
Биссектриса угла — это луч, который делит угол пополам. Из вершины \(B\) проведена биссектриса \(BD\) угла \(\angle ABC\), равного \(106^\circ\). Значит каждая из половинок угла имеет меру
\[
\angle ABD = \frac{106^\circ}{2} = 53^\circ.
\]

Шаг 4.  
По определению угол между двумя лучами, выходящими из одной точки, равен разности (или модулю разности) углов, которые эти лучи образуют с каким-то общим лучом. Мы уже нашли:
\(\angle ABH = 80^\circ\) (между \(BA\) и \(BH\)) и \(\angle ABD = 53^\circ\) (между \(BA\) и \(BD\)). Тогда угол между \(BH\) и \(BD\) при вершине \(B\) равен
\[
\angle HBD = |\angle ABH - \angle ABD| = |80^\circ - 53^\circ| = 27^\circ.
\]
Это и есть искомый угол.
