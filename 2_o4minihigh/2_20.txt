Шаг 1. $x(x^4 - 9x^2 + 20) = 0$.  
Шаг 2. По свойству нулевого произведения либо $x = 0$, либо $x^4 - 9x^2 + 20 = 0$.  
Шаг 3. В уравнении $x^4 - 9x^2 + 20 = 0$ сделаем замену $t = x^2$. Получим квадратное уравнение  
\[
t^2 - 9t + 20 = 0.
\]  
Вычисляем дискриминант $D = (-9)^2 - 4 \cdot 1 \cdot 20 = 1$, тогда  
\[
t_{1,2} = \frac{9 \pm 1}{2} \;\Longrightarrow\; t_1 = 5,\; t_2 = 4.
\]  
Шаг 4. Возвращаемся к $x$:  
\[
x^2 = 5 \;\Longrightarrow\; x = \pm\sqrt{5}, 
\qquad
x^2 = 4 \;\Longrightarrow\; x = \pm 2.
\]  
Шаг 5. Итак, все корни исходного уравнения:
\[
x \in \{\,0,\;2,\;-2,\;\sqrt5,\;-\sqrt5\}.
\]

solutiontextexpanded:
Шаг 1. Мы видим, что каждое слагаемое в многочлене $x^5 - 9x^3 + 20x$ содержит общий множитель $x$. Чтобы упростить уравнение, выносим этот множитель за скобки. По правилу распределительного свойства:  
\[
x^5 - 9x^3 + 20x = x\bigl(x^4 - 9x^2 + 20\bigr).
\]
Это позволяет нам разбить одну сложную запись на произведение двух множителей.

Шаг 2. Теперь у нас стоит уравнение вида $x\cdot P(x) = 0$, где $P(x)=x^4-9x^2+20$. По свойству нулевого произведения, если произведение двух множителей равно нулю, то хотя бы один из них равен нулю. Значит, получаем два случая:
1) $x = 0$,
2) $x^4 - 9x^2 + 20 = 0$.

Шаг 3. Второе уравнение — это уравнение четвертой степени. Чтобы превратить его в более простое, вводим новую переменную $t = x^2$. Так как $x^2$ неотрицательно, $t\ge0$, но для решения квадратного уравнения нам это не помешает. Подстановка даёт:
\[
x^4 - 9x^2 + 20 = t^2 - 9t + 20 = 0.
\]
Это уже классическое квадратное уравнение. Для его решения находим дискриминант:
\[
D = b^2 - 4ac = (-9)^2 - 4\cdot1\cdot20 = 81 - 80 = 1.
\]
Корни уравнения по формуле $t = \frac{-b \pm \sqrt{D}}{2a}$:
\[
t_1 = \frac{9 + 1}{2} = 5,\qquad t_2 = \frac{9 - 1}{2} = 4.
\]

Шаг 4. Теперь возвращаемся к исходной переменной $x$:  
– Если $x^2 = t_1 = 5$, то $x = \pm\sqrt{5}$.  
– Если $x^2 = t_2 = 4$, то $x = \pm2$.  
Здесь мы используем свойство корня: если $x^2 = a\ge0$, то $x = \pm\sqrt{a}$.

Шаг 5. Собирая все найденные корни, получаем полный список решений исходного уравнения:
\[
x = 0,\; x = 2,\; x = -2,\; x = \sqrt{5},\; x = -\sqrt{5}.
\]
